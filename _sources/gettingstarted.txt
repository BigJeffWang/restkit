.. _gettingstarted:

Getting Started
===============

This tutoriall exposes key features of this library mainly through code
examples. For in-depth description of the modules, you'll want to read 
the :ref:`API <api>` documentation.

If you have not installed restkit yet, the :ref:`installation docs <installation>`
will get you started. 

Perform HTTP call support  with `restkit.rest.Resource`.
-----------------------------------------------------------
Usage example, get friendpaste page :


.. code-block:: python

    from restkit import Resource
    res = Resource('http://friendpaste.com')
    page = res.get()
    print page.body
    print page.status_int_

Perform HTTP call support  with `restkit.rest.RestClient`.
-------------------------------------------------------------

Usage example, get friendpaste page:


.. code-block:: python

    from restkit import RestClient
    client = RestClient()
    page = resource.request('GET', 'http://friendpaste.com')

Or get a paste in JSON :

.. code-block:: python

    from restkit import RestClient
    client = RestClient()
    client.request('GET', 
        'http://friendpaste.com/5rOqE9XTz7lccLgZoQS4IP'),
    headers={'Accept': 'application/json'})


Basic Authentification:
------------------------

.. code-block:: python

    from restkit import RestClient
    from restkit import httpc
    transport = httpc.HttpClient()
    transport.add_authorization(httpc.BasicAuth("username", "password"))
    client = RestClient(transport=transport)
    user_timeline = client.request('GET', "http://twitter.com/statuses/user_timeline.json")
    print user_timeline.body
